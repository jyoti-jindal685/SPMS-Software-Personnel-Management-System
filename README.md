💼 Software Personnel Management System (SPMS)

The Software Personnel Management System (SPMS) is a web-based frontend application designed to demonstrate the interaction between HR and employees within an organization. It provides a structured and user-friendly interface for managing employee-related information and job listings.

---

🚀 Features

- 👨‍💼 Employee Interface
- 🧑‍💻 HR Dashboard UI
- 📋 Job Listings Page
- 🎯 Clean and Responsive Design
- 📊 Organized Information Display

---

🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript

---

📂 Project Structure

SPMS/
│── index.html
│── jobs.html
│── style.css
│── script.js
│── README.md

---

⚙️ How to Run

1. Clone the repository:

git clone https://github.com/jyoti-jindal685/SPMS-Software-Personnel-Management-System.git

2. Open the project folder:

cd SPMS-Software-Personnel-Management-System

3. Run the project:
   Open "index.html" in your browser
